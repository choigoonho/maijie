DMX-512 Signals Amplifier / Splitter
     
   
     
 

The DMX-512 distributor unit has two input and some outputs. Absolutely electric isolation between input and output or between output and output. Two type of DMX-512 connector(3pin) for each individual channel, each DMX-512 output features a independent driver.

Power input: 120V AC / 230V AC
