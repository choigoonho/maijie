2.4G Wireless DMX Transmiter 
 
           
       
           
 

This is a sort of Wireless DMX Controller (Signal Emission) to receive the DMX signal from DMX console and emit the signal to another Wireless DMX Controller (Signal Receipt) to control the fixture in DMX signal.

  
           
 

Model Name：512D-W2400
Input Power：9V/500MA
Transmission Distance：400M (Open field)
Working Frequency：2400M～2431M(2.4G ISM)
Transmission Power：20DBM

      
           
   
           
   
           
 

①  Antenna SMA Connector
②  DC input outlet
③  DMX signal indicator light
④ Wireless Remote indicator light: Signal Receipt (RED)、Signal Emission (Green)
⑤ The Dip Switch 1, code No.1-No.4 for setting up the connection address for Signal Receipt controller when more than 2pcs are connected (real time setting available)
⑥ The Dip Switch 2, code No.1-No.5 are for setting up of choosing the working frequency, No.6 is the On-Off switch of the signal receipt and signal emission (ON is for signal emission, OFF is for signal receipt, reset need to cut off power first) (real time setting available)
