MENU Mode:

                  
                         
   

Addr :  DMX mode & Auto Mode .Choose DMX512 address A001~A512.
blUE :  Blue dimmer b000~b255.
grEE :  Green dimmer g000~g255
rEd :   Red dimmer r000~r255
diSP :  yes or no to choose the menu clockwise or anti-clockwise
SPEE :   Setup Auto run speed
AutO :   22 type auto run mode (Pr01~Pr22)
SOUN :  Enter sound active mode
